# Vector Embeddings Exercise

This exercise demonstrates how to generate and work with vector embeddings using Azure OpenAI and Azure Cognitive Search for the RAG-enabled chatbot workshop.

## Overview

In this exercise, you'll learn how to:
- Generate text embeddings using Azure OpenAI
- Store and manage vector embeddings in Azure Cognitive Search
- Implement vector similarity search
- Process and normalize embedding results

## Prerequisites

- Azure Subscription
- Azure OpenAI Service access
- Azure Cognitive Search instance
- .NET 6.0 SDK
- Visual Studio 2022 or VS Code

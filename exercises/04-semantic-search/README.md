# Semantic Search Exercise

This exercise demonstrates how to implement semantic search capabilities using Azure Cognitive Search and Azure OpenAI for the RAG-enabled chatbot workshop.

## Overview

Learn how to:
- Implement semantic search with vector capabilities
- Process and optimize search queries
- Handle and format search results
- Combine traditional and vector search approaches

## Prerequisites

- Azure Subscription
- Azure OpenAI Service access
- Azure Cognitive Search instance
- .NET 6.0 SDK
- Visual Studio 2022 or VS Code

# Echo Bot Exercise

This exercise demonstrates the basic functionality of the Microsoft Bot Framework.

## Prerequisites
- .NET 6.0 SDK
- Visual Studio 2022 or VS Code
- Bot Framework Emulator

## Setup
1. Clone this repository
2. Open the solution in your IDE
3. Build and run the project
4. Connect using Bot Framework Emulator

## Features
- Echoes back user messages
- Sends welcome message to new users
- Includes basic error handling

## Testing
1. Start the bot
2. Open Bot Framework Emulator
3. Connect to http://localhost:3978/api/messages
4. Start chatting with the bot

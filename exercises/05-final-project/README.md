# RAG-Enabled Chatbot with Microsoft Bot Framework

A production-ready implementation of a Retrieval-Augmented Generation (RAG) chatbot using the Microsoft Bot Framework, Azure OpenAI, and Azure Cognitive Search.

## Overview

This chatbot demonstrates:
- Document processing and text extraction
- Vector embeddings generation
- Semantic search capabilities
- Natural language response generation
- Integration with Azure services

## Prerequisites

- Visual Studio 2022 or VS Code
- .NET 6.0 SDK
- Azure Subscription with access to:
  - Azure OpenAI Service
  - Azure Cognitive Search
  - Azure Form Recognizer
- Bot Framework Emulator
